# 项目简介

AI 前端监控平台是一个面向现代前端应用的监控解决方案，旨在帮助开发者快速发现和解决前端问题。

## 设计理念

### 1. 微服务架构

项目采用 Monorepo 架构，将不同功能模块分离为独立的包：

- **metrics**: 前端监控 SDK，负责数据采集
- **ai**: 后端分析引擎，负责智能分析
- **doc**: 项目文档，提供完整的使用指南

### 2. 可插拔设计

每个包都可以独立使用，也可以组合使用：

- 只需要基础监控？使用 `metrics` 包
- 需要智能分析？组合使用 `metrics` + `ai`
- 需要完整功能？使用全部包

### 3. 开发者友好

- 完整的 TypeScript 支持
- 详细的类型定义
- 丰富的文档和示例

## 技术选型

| 包名    | 技术栈                      | 用途             |
| ------- | --------------------------- | ---------------- |
| metrics | Vite + TypeScript + rrweb   | 前端数据采集 SDK |
| ai      | NestJS + LangChain + Ollama | 后端智能分析引擎 |
| doc     | RSPress + React             | 项目文档         |
| example | Vite + React                | 示例应用         |
